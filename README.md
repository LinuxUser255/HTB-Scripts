A personal collection of Bash and Python scripts useful for **Hack The Box**, penetration testing, Active Directory enumeration, credential hunting, and post-exploitation tasks.

## Features

- Organized by category such as OS, Technology, Web, etc..
- Focused on common HTB scenarios (AD, SMB enumeration, scanning, credential extraction)
- Reusable, well-commented scripts
- GPL v3 licensed
